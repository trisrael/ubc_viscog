Help

This program will run a visualization experiment.

Configuring an experiment

To configure an experiment press the 'Configure Experiment' button. This will bring up the experiment configuration window. To edit a configuration double click on any cell in the 'value' column, and enter a valid value. Basic error checking will prevent some obvious errors from being made, however it should be ensured that values entered make sense.

To save the configuration to file, press the 'Save' button. To ignore any changes press the 'Cancel' button.

Running test trials

To run subjects in easy test trials, press the 'Test Experiment' button. Preset conditions will be run, with the number of trials per test condition being configurable by the configuration window.

Press the 'Esc' key to close the test at any time.

Running the experiment

Press 'Run Experiment' to run the experiment. All conditions in the preset conditions file will be run.

Prior to running the experiment, please enter the subject's initials and number (which can be found on the experiment consent form). The initials will be used to help the RA analyzing the experiment data, and the subject number will be used to counterbalance the experiment, if counterbalancing is desired.

Press the 'Esc' key to close the experiment at any time.
